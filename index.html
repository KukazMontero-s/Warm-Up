Informaciones

• Fondo Previsional

Este primer simulador se encarga de entregar un fondo previsional, de acuerdo, a un sueldo fijo y una rentabilidad constante en el tiempo, siendo estas:

Fondo
  • A -> 7.27%
  • B -> 6.18%
  • C -> 4.74%
  • D -> 2.57%
  • E -> 1.43%

En el caso de que el usuario no esté conforme con estos valores de rentabilidad, se da la opción de que el usuario ingrese una rentabilidad.

Por otro lado tenemos la edad de jubilacion, la cual por defecto es 60 ó 65 dependiendo del sexo ingresado, para el calculo se toman en cuenta la edad de jubilacion (si es ingresada) por sobre el sexo ingresado, y se calcula los años desde su edad actual hasta la edad de jubilacion/sexo

Por ultimo cabe destacar que la variable "APV Total" hace referencia a el APV que se ingresara durante todo este periodo de años, el total.

• APV necesario

Para este caso tenemos algo bastante parecido, de hecho se calcula el fondo previsional (sin APV), con el fin de obtener el APV necesario para alcanzar el "Fondo esperado a jubilar", este valor se entrega de manera Total y anual para una una mayor comprensión por parte del usuario.
